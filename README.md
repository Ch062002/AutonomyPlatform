# Aerospace Autonomy Platform

Software-first aerospace autonomy infrastructure platform focused on:

- UAV mission systems
- Guidance and control
- PX4/ROS2 integration
- Simulation infrastructure
- Swarm autonomy
- AI-assisted mission intelligence

## Core Technologies

- Ubuntu 24.04
- ROS2 Jazzy
- PX4
- Gazebo
- FastAPI
- React
- Docker
- Python
